Step 5: Add Your Content

This is where the words go — the story, the pitch, the message. Let’s write the site.

Start With the Core Pages

  • Home: Hook, pitch, preview of what’s inside
  • About: Your story, credentials, why you
  • Services / Products: Clear breakdown with calls to action
  • Contact: Email, form, phone, map
  • FAQs: Answer objections before they’re asked

🧠 Plan Your Structure

Use folders and file names that make sense for users and search engines.

  • /about.html
  • /services/index.html
  • /contact/index.html

✍️ Write Like a Human

Use simple language, short sentences, and action verbs. Every page should pass the "So what?" test.

  • Write for clarity, not cleverness
  • Break up text with subheadings and lists
  • Use strong CTAs like “Get a Quote” or “Call Now”

🔍 SEO Considerations

Each page should target 1–2 keywords naturally, include meta tags, and have meaningful headings.

  • Use <h1> only once per page
  • Use alt tags on every image
  • Structure content with semantic HTML

📸 Add Real Media

Stock photos are fine, but real images convert better. Use TinyJPG to compress them for fast load times.

  • JPEG for photos, PNG for graphics, SVG for icons
  • Keep image file sizes under 500 KB
  • Always include descriptive filenames (e.g. plumber-working.jpg)

Copywriting Tools

  • Grammarly — for spelling and tone
  • Headlime or Copy.ai — AI copy helpers
  • CoffeeShop Copywriters — examples of compelling writing
